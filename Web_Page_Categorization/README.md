# Web_Page_Categorization

Program reads in a control file which contains Wikipedia URL. For each URL, the page word frequencies and other metrics are gathered. User enters any valid wikipedia URL and the program provides a URL to the most closely related known page.

Key implementations in this project:
* JavaFX GUI
* Resizeable and persistent HashTable
* Webscraping with Jsoup
* tf-idf (Term frequency-inverse document frequency) Similarity metrics
